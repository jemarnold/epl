# Parvo 30-sec binned export of maximal graded cycling assessment

An .xlsx file exported from Parvo Medics TrueOne 2400 metabolic cart
with 30-sec binned average ventilation, gas exchange, and cycling work
rate & cadence data for a maximal incremental ramp to V̇O~2~peak.

## Format

A raw .xlsx file with metadata and a primary data table with 20 columns
and 55 rows.

## Details

Session records a maximal incremental ramp to V̇O~2~. Protocol details
are unknown. Includes work rate (*"WorkR"*) and cadence (*"RPM"*).
